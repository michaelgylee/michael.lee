# AI Card News Agent Team (AI 카드뉴스 에이전트 팀)

인공지능 에이전트 협업 팀을 구축하여 최신 AI 기술 소식을 자동으로 수집(Research)하고, 보기 좋은 카드뉴스(1:1 비율) 형태로 제작(Create)하며, 카카오톡 배포용 규격으로 완성(Publish)하는 종합 자동화 프로젝트입니다.

이 프로젝트는 터미널 기반의 **Python CLI 자동화 스크립트**와 실시간 프리뷰 및 상세 편집이 가능한 **프리미엄 웹 대시보드** 두 가지 방식을 모두 지원합니다.

---

## 📂 프로젝트 구조 (Project Structure)

```text
kakao_cardnews_agent/

├── index.css               # 대시보드 CSS 스타일시트 (Glassmorphism & Neon theme)
└── app.js                  # 대시보드 에이전트 파이프라인 및 내보내기 JS 로직
```

## 💻 1. 웹 대시보드 구동 가이드 (Web Dashboard)

웹 대시보드는 세 에이전트의 작동 과정을 시각적으로 확인하고, 생성된 카드뉴스를 브라우저에서 직접 수정 및 다운로드할 수 있는 프리미엄 관리 도구입니다.

### 실행 방법
1. 프로젝트 폴더가 있는 위치에서 간단한 로컬 웹 서버를 구동합니다.
   ```bash
   python3 -m http.server 8000
   ```
2. 웹 브라우저를 열고 `http://localhost:8000`에 접속합니다.

### 주요 기능
- **에이전트 워크플로 시각화**: 가동 시 조사(Researcher) -> 제작(Creator) -> 배포(Publisher) 에이전트가 데이터와 신호를 주고받는 애니메이션을 확인할 수 있습니다.
- **실시간 카드 편집기**: 생성된 슬라이드 카드를 클릭하면 본문 내용, 제목, 글자 크기, 백그라운드 그라데이션Preset을 실시간으로 커스텀할 수 있습니다.
- **고해상도 다운로드 (html2canvas & JSZip)**: 모든 수정 완료 후 `카드뉴스 다운로드 (ZIP)` 버튼을 누르면 최대 10장의 슬라이드를 `1800x1800px`의 고화질 이미지 패키지로 압축하여 자동 다운로드합니다.
- **Gemini API 연동**: 설정창에 Google Gemini API Key를 입력하면, 시뮬레이션 데이터를 넘어 구글 뉴스 실시간 파싱 결과를 Gemini가 직접 분석하여 실시간 카드뉴스를 완전 자동 제작합니다.
- **한국시간·발행일 정합성**: 모든 모드는 `Asia/Seoul` 기준으로 수집 기간을 계산하며, 카드 제목에는 생성 요청일이 아니라 각 원문의 실제 발행일을 표시합니다.
- **최신 소스 보호**: 일간/주간 Google News RSS와 TreeSoop 최신 게시물을 매 실행 시 새로 조회합니다. Google News 기본 경로가 실패하면 경량 RSS JSON으로 자동 전환합니다.
- **동일 비율 고해상도 출력**: 240×240 미리보기 전체를 동일 배율로 720×720 렌더링한 뒤 2.5배 래스터화하여, 제목·본문·로고·여백 비율이 실제 JPEG에서도 유지됩니다.
- **v3.1.7 Litify TL;DR**: 긴 기사 본문을 3~4개의 짧은 핵심 불릿으로 재구성합니다. 본문 카드 하단에는 대표 사이트명과 뉴스 업로드 날짜만 표시하되 웹에서는 원문 링크로 동작하며, 마지막 카드에는 실제 카카오톡 채널 URL을 노출합니다.
- **v3.1.8 Compact & Resilient**: 본문 항목 번호와 제목 날짜를 제거하고 완결된 짧은 문장으로 압축해 잘림을 방지합니다. 공개 CORS 프록시의 503/413 오류 시 검증된 내장 캐시로 자동 복구하며, 에이전트 상태 카드에 실제 수집·재시도·검증 진행률을 표시합니다. 3대 LLM 업데이트 옵션은 기본 활성화됩니다.
- **v3.1.9 Scrollable & Clean Sources**: 슬라이드 프리뷰를 모든 화면에서 가로 스크롤할 수 있게 개선하고, TreeSoop 원문 URL이 본문 불릿으로 섞이는 문제와 Source 푸터 잘림을 수정했습니다. RSS 제목만 확보된 경우 조사 안내 문구를 만들지 않고 실제 헤드라인만 표시합니다. 브라우저에서 최신 원문을 안정적으로 읽을 수 없는 Trend Chaser 생성 기능은 제거했습니다.
- **v3.2 Adaptive Layout & Quality Retry**: 일간·주간·TreeSoop 카드에 콘텐츠 높이 기반 자동 폰트 축소와 Source 안전 영역 검증을 공통 적용했습니다. URL의 마침표 때문에 문장이 분리되는 오류를 막고 짧거나 미완결된 후보 뉴스는 교체합니다. 문장 완결성·정보량·번역 및 레이아웃 검증에 실패하면 검증 에이전트가 제작 에이전트에 최대 2회 재작성을 요청하며 전체 결과는 10장 이내로 제한됩니다.
- **v3.2.1 TreeSoop Pipeline Hotfix**: TreeSoop의 정상적인 장문 불릿을 82자 고정 기준으로 반복 반려하던 문제를 제거하고 실제 렌더링 안전 영역으로 판정합니다. 품질 재작성 중에는 제작 에이전트 진행률만 갱신해 제작 45%·검증 35%에서 멈춘 것처럼 보이던 상태 표시도 수정했습니다.
- **v3.2.2 Mobile Fast News Build**: iOS Safari의 잘못된 안전영역 판정과 고메모리 이미지 내보내기를 수정했습니다. 카카오톡 미리보기·공유 설정과 중복 검증 UI를 제거하고 검증 에이전트를 하나로 통일했으며, 경고가 결과 출력을 차단하지 않도록 자동 보정·안전 대체 출력을 적용했습니다. 일간·주간 RSS 카드에는 3개의 핵심 불릿을 보장하고 인위적 대기시간과 소스 연결 타임아웃을 단축했습니다.
- **v3.2.3 Fact Summary & Theme Build**: 일간·주간 RSS 카드가 다른 기사를 섞거나 날짜·원문 안내 문장으로 불릿을 채우지 않도록 수정했습니다. 한 기사 제목에서 주체·핵심 제품/주제·행동을 분리한 3개 사실 불릿을 생성하며, 전체 카드에 적용되는 성아연 로고 기반 기본·산뜻한 그라디언트·블랙 테마 선택기를 추가했습니다.
- **v3.2.4 Original Source Summary Build**: 제목을 반복하거나 주체·키워드만 나열하던 RSS 요약을 폐기했습니다. GitHub Actions가 Google News 링크를 발행사 원문으로 해석하고 본문에서 검증된 3개 완결 문장을 추출해 3시간마다 정적 캐시를 갱신하며, 웹은 이 캐시만 카드 후보로 사용합니다. TreeSoop은 표지·마지막 장을 제외하고 최신 일자 게시물에 실제로 파싱된 뉴스 수만큼 카드를 생성합니다.

## 🐍 2. Python CLI 구동 가이드 (CLI Agent Runner)

서버 백그라운드 환경 등에서 카드뉴스 제작을 자동화하고 실제 이미지 파일로 출력하는 Python 스크립트입니다.

### 실행 방법
1. 필수 라이브러리를 설치합니다.
   ```bash
   pip3 install -r requirements.txt
   ```
2. (선택사항) 실시간 AI 요약을 원할 경우 동일 경로에 `.env` 파일을 생성하고 Gemini API 키를 저장합니다.
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```
3. 에이전트 스크립트를 실행합니다. (주간 카드뉴스를 원할 경우 인자로 `weekly` 전달)
   ```bash
   # 일간 카드뉴스 생성
   python3 agent_runner.py
   
   # 주간 카드뉴스 생성
   python3 agent_runner.py weekly
   ```
4. 실행 완료 시 `output/` 폴더 아래 7장의 카드뉴스 이미지(`01_slide.jpg` ~ `07_slide.jpg`) 및 카카오톡 배포용 스키마 JSON 파일(`kakao_payload.json`)이 자동 저장됩니다.

> 최신 뉴스 수집에는 인터넷 연결이 필요합니다. 실제 발행일을 확인할 수 없는 경우에는 잘못된 최신 카드뉴스 생성을 막기 위해 명확한 오류와 함께 작업을 중단합니다.
